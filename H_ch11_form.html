Label is used for define a text description for an input element in an HTML form. It tells the user what information should be entered in a particular input field.
The for attribute connects the label to an input using the input’s id.

Important attribute

  • Action : It is an attribute . it is the address at which the from is submitted after filling.
  • Method : Type of the data (sensitive or insenitive)
  • encrypt : enctype is an attribute of the form tag that specifies how form data should be encoded before being sent to the server.
  • required : If we write 'required' attribute then it is mandatory to fill that section.
  • minlength : If we write minlength="" attribute in the text then it is mandatory to have equal or more than given no of characters.
  • maxlength : If we write maxlength="" attribute in the text then it is mandatory to have equal or less than given no of characters.
  • placeholder : The placeholder attribute is used inside form input fields to display a short hint or example text that tells the user what kind of data should be entered.
  • min/max : The min and max attributes are used in form input elements to define the minimum and maximum allowed values that a user can enter
  • value : The value attribute specifies the default data of an input field that is sent when the form is submitted.
  • name : The name attribute is used to identify an input field in an HTML form. It defines the key (variable name) that is sent to the server along with the input’s value when the form is submitted.
  • accept : The accept attribute is used with input type="file" to specify the types of files that the user is allowed to upload.
